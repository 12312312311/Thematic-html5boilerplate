Thematic-html5boilerplate by sams

A Thematic theme for Wordpress using the goodness of HTML5 Boilerplate

HTML5Boilerplate 0.3a (pushing soon) The Pre Beta

Features

More support for more plugins; additional controls for elements that are often customised; better use of wp hooks and more power in general.

  • a gallery (basic and highly customisable jquery powered gallery)
  • use of PHPThumb by Master Exploder
  • tighter preset styles (within liquid options and grid debug)
  • Out of the Zip support for popular and powerful wp-plugins such as Minify, WP-Supercache & White Label CMS
  • Childthemes (customise the theme as a child of this theme to make upgrades easier)
  • Moving towards the inclusion of Brian Blakelys head elements
  • Scores well in YSLow and Google Page Speed; these results have been confirmed by Vertain and Website Optimization Analysis Tools

Combing the html5 boilerplate, OOCSS & wordpress. With builtin support for a whole verity of plugins - and more

Install

Install the latest tagged version as per normal wordpress theme installion instructions
